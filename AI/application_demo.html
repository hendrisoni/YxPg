Key Core Features

Interactive Result Grid Updated

Powered by Tabulator for fast, customizable rendering of large query results. Features column reordering, resizing, and instant filtering.

Advanced Column Filter New UI

A smart toggling header filter integrated directly into the left-most column header (#). Click the funnel icon in the index column header to instantly show/hide column-level filters.

High-Speed XLSX Export New

Download query result sets directly into premium formatted Microsoft Excel (.xlsx) files. The row index is automatically excluded, and columns output clean text headers instead of HTML.

SQL Editor with Darcula Theme Updated

An advanced CodeMirror 6 query editor featuring autocomplete, syntax highlighting, syntax matching, and a highly readable custom dark-blue text selection highlight.
